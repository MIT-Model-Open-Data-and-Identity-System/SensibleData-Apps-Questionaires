About

What?

SensibleDTU: Questionnaires is part of Sensible DTU project. It is an application for filling out questionnaires.

The back-end of this application enables creation of online surveys with no need for any programming. The current feature set is:

  • Generation of online surveys from an Excel spreadsheet
  • Dynamic saving of the answers - the participant can drop the survey at any point and come back to where they left later
  • Input validation
  • Support for a number of question types: single choice, multiple choice, number/time inputs, grid questions, free text, and more
  • It's free and open-source!
